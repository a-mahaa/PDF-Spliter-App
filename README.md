# PDF-Spliter-App

Description:
This project contains multiple PDF utilities built with Python. The program can split PDF pages, merge PDF files, and convert PDF text into an audiobook MP3 file.

Language Used:
Python

Libraries Used:
PyPDF2
gTTS
os

Additional Files Required:
This project requires PDF files such as "file1.pdf", "file2.pdf", and "Cover.pdf" to run the different PDF features.

Features:
- Split specific PDF pages
- Merge multiple PDF files
- Convert PDF text into speech
- Save audiobook as MP3
- Display saved file location
